# 音乐

Taylor Swift - Speak Now (Taylor's Version)

Jiangsu,China


《爱的告白 (重制版)》（英语：Speak Now (Taylor’s Version)）是美国创作歌手泰勒·斯威夫特的第三张重录专辑，于2023年7月7日由联众唱片发行。这是继她于2021年4月发行的第一张重录专辑《放手去爱 (重制版)》和2021年11月发行的第二张重录专辑《红 (重制版)》之后，对她的第三张录音室专辑《爱的告白》（2010年）的重新录制。重新录制是斯威夫特对她前六张录音室专辑的母带所有权改变的反制措施。
